
# Truck-to-Rail Conversion Intelligence MVP

## Run locally

1. Create virtual environment

Windows:
python -m venv venv

Mac/Linux:
python3 -m venv venv

2. Activate

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

3. Install requirements

pip install -r requirements.txt

4. Run app

streamlit run app.py
